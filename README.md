# 知识星球内容下载工具

从知识星球"天上不会掉馅饼"（group ID: 28888825825151）下载许哲的内容和评论，保存为 Markdown 文件。

## 功能特性

- ✅ 下载许哲发布的所有主题帖
- ✅ 下载许哲在其他人帖子下的评论和回复
- ✅ 自动下载图片和附件到本地
- ✅ 生成格式化的 Markdown 文件
- ✅ 支持全量下载和增量更新
- ✅ 智能检测新评论并更新
- ✅ **断点续传** - 全量下载中断后可继续，不会重复下载
- ✅ **实时进度显示** - 进度条、已用时间、预估剩余时间一目了然
- ✅ **分批下载** - 可指定每次下载数量，灵活控制下载进度

## 安装

项目依赖已安装，包括：
- `playwright` - 浏览器自动化
- `date-fns` - 日期处理
- `typescript` - TypeScript 支持
- `tsx` - TypeScript 执行器

## 使用方法

### 方式 1: 使用 Claude Code Skill（推荐）

在 Claude Code 中使用 `/xz` 命令：

```bash
# 增量更新（默认）- 检查最新 20 条帖子的更新
/xz

# 或明确指定增量模式
/xz update

# 全量下载 - 下载所有内容
/xz full

# 分批下载 - 每次下载指定数量
/xz full 10    # 每次下载10个
/xz 10         # 简写方式
/xz full 20    # 每次下载20个
```

### 方式 2: 使用 npm 命令

```bash
# 增量更新
npm run sync:update

# 全量下载
npm run sync:full
```

### 方式 3: 直接运行

```bash
# 增量更新
npx tsx src/index.ts --mode=incremental

# 全量下载
npx tsx src/index.ts --mode=full
```

## 首次使用

1. 首次运行时，浏览器会自动打开
2. 如果未登录，请在浏览器中完成登录
3. 登录成功后，工具会自动开始抓取内容
4. 抓取完成后，检查 `output/` 目录中的 Markdown 文件

## 工作原理

1. **浏览器自动化**: 使用 Playwright 打开知识星球网页
2. **网络拦截**: 监听并拦截知识星球的 API 请求
3. **数据提取**: 从 API 响应中提取帖子数据
4. **内容筛选**: 筛选出许哲相关的内容（他发布的帖子和评论）
5. **下载附件**: 下载图片和文件到本地
6. **生成 Markdown**: 将内容转换为格式化的 Markdown
7. **状态管理**: 记录同步状态，支持增量更新

## 同步模式

### 全量下载 (`full`)

- 下载所有包含许哲内容的帖子
- 适合首次使用
- 耗时较长

### 增量更新 (`update`)

- 仅检查最新 20 个帖子
- 检测新帖子和新评论
- 适合日常使用
- 速度快

### 分批下载 (`full [数量]`)

- 每次下载指定数量的帖子
- 适合时间有限或想分批完成
- 多次运行自动继续下载剩余内容
- 灵活控制每次下载进度

**示例：**
```bash
/xz full 10    # 每次下载10个
/xz 10         # 简写方式
```

详细说明请查看 [BATCH_DOWNLOAD.md](BATCH_DOWNLOAD.md)

## 断点续传

全量下载支持断点续传功能，如果下载过程中中断（网络问题、手动停止等），下次运行时会自动从上次中断的位置继续：

**工作原理：**
- 下载过程中自动保存进度（已下载到的最早时间戳）
- 检测到未完成的下载时，自动从上次位置继续
- 不会重复下载已保存的内容

**使用方式：**
```bash
# 首次运行全量下载
/xz full

# 如果中断了，再次运行相同命令即可继续
/xz full  # 会自动检测并继续
```

**进度显示：**
```
⚠️  检测到未完成的下载
📍 将从上次中断位置继续（最早时间: 2026-01-15T10:00:00Z）
📊 已下载: 150 个帖子
```

## 注意事项

1. **登录状态**: 首次使用需要在浏览器中登录知识星球
2. **网络速度**: 下载速度取决于网络连接
3. **浏览器窗口**: 浏览器会保持打开状态，方便调试
4. **频率限制**: 建议不要过于频繁地运行全量下载

## 常见问题

### 1. 浏览器无法打开？

确保已安装 Playwright 浏览器：

```bash
npx playwright install chromium
```

### 2. 登录后自动退出？

可能需要手动保持浏览器打开状态，或者使用持久化浏览器上下文。

### 3. 图片下载失败？

某些图片可能有访问限制，失败时会保留原始 URL。

### 4. 找不到许哲的内容？

检查配置文件中的 `target_author` 是否正确。

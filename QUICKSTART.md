# 快速启动指南

## 三步开始使用

### 步骤 1: 安装 Playwright 浏览器

```bash
npx playwright install chromium
```

这只需要运行一次，大约需要下载 300MB。

### 步骤 2: 首次全量下载

使用 Claude Code Skill：

```bash
/xz full
```

或使用 npm：

```bash
npm run sync:full
```

**会发生什么：**
1. 浏览器自动打开
2. 如果未登录，请在浏览器中登录知识星球
3. 登录后，工具自动开始抓取内容
4. 实时显示进度（进度条、已用时间、预估剩余时间）
5. 等待完成（可能需要几分钟）
6. 在 `output/` 目录查看生成的 Markdown 文件

**进度显示示例：**
```
[50/156] 处理: 关于时间管理的建议...
总进度: 50/156 [█████████░░░░░░░░░░░░░░░░░░░░░] 32% | 已用时: 2分30秒 | 预计剩余: 5分18秒
```

详细示例请查看 [PROGRESS_EXAMPLE.md](PROGRESS_EXAMPLE.md)

### 步骤 3: 日常增量更新

```bash
/xz
```

这将：
- 只检查最新 20 个帖子
- 下载新帖子和有新评论的帖子
- 速度快，适合定期运行

## 进阶使用：分批下载

如果想分多次完成下载，可以使用分批模式：

```bash
# 每次只下载 10 个帖子
/xz 10

# 或者明确指定
/xz full 10
```

**优势：**
- ⏱️ 时间可控，不会一次占用太长时间
- 🔄 可以随时停止，下次继续
- 📊 每次完成后显示还剩多少

详细说明请查看 [BATCH_DOWNLOAD.md](BATCH_DOWNLOAD.md)

## 就这么简单！

现在你可以：
- 在 `output/` 目录查看所有下载的内容
- 用任何 Markdown 编辑器打开文件
- 图片和附件都在本地，可以离线查看

## 进度条说明

### 符号含义

- `█` - 已完成部分
- `░` - 未完成部分

### 百分比计算

```
当前进度 / 总数 × 100%
```

### 时间预估算法

```
已用时间 / 已处理数量 = 平均每项耗时
平均每项耗时 × 剩余数量 = 预估剩余时间
```

## 实时更新

进度信息会在每个帖子处理完成后立即更新，让您随时了解：
- 当前处理到哪一个
- 还剩多少没处理
- 大概还需要多久

这样即使下载大量内容，也能清楚地知道进度！
